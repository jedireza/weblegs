Installation:

  1. Create new mysql database and import the mysql-db-structure.sql file.
  2. Configure project settings via the app/config.php file. Make sure you supply MySQL credentials. Supply SMTP credentials if you want emails to send.
  3. Tell Apache to serve files from www/.

Gotchas:

  1. You'll need PHP's libxsl module installed. We use version 1.1.26 compiled against libxml version 2.7.8.
  2. Don't install as directory under another domain, setup your project as it's own domain.
